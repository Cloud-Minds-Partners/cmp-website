    Patch src/layouts/Base.astro: add `<slot name="head" />` inside `<head>`, immediately before `</head>`. Without this slot, SEO component output goes to `<body>` (Pitfall 5 from RESEARCH.md).

    The patched head section must look like:
    ```
    <head>
      <meta charset="UTF-8" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <!-- ... existing meta tags preserved verbatim ... -->
      <title>{fullTitle}</title>
      <slot name="head" />
    </head>
    ```

    Also remove the current hardcoded meta tags that SEO.astro will manage (og:title, og:description, og:type, og:site_name, meta description, meta robots, canonical) — BUT keep the charset, viewport, color-scheme, theme-color, and favicon.
